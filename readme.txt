== Description ==

Interactive posts is a new plugin allowing users to create basic quizzes, questions and interactions as part of a WordPress site.

For support please visit http://www.pgogy.com and http://www.pgogy.com/code/groups/wordpress/interactive-posts/

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
